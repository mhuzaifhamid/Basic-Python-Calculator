***Basic Python Calculator***

A simple ***console-based calculator*** built in Python that supports basic arithmetic operations: addition, subtraction, multiplication, and division.  

---

## Features
- Addition ➕
- Subtraction ➖
- Multiplication ✖️
- Division ➗ (handles division by zero)
- Console-based user interface

---

## How to Use
1. Download or clone this repository.
2. Open the `calculator.py` file in your preferred editor (e.g., VS Code, PyCharm, Sublime Text, or IDLE).
3. Run the file.
4. Follow the on-screen instructions to perform addition, subtraction, multiplication, or division.
5. To exit the calculator, choose option 5 or type 'n' when asked to continue.
